# Jatos Builder
### Create certs
`cd devops/traefik/certs/` 
`./install.sh`
`./create.sh`

### Start/stop proyect
`./start.sh`
`./stop.sh`

### URLs
- App `https://app.docker.localhost`
- Jatos `https://jatos.docker.localhost`
### Defaults Users
- App 
	- User `admin@local.host`
	- Password `secret`
- Jatos
	- User `admin`
	- Passowrd `admin`
